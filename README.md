# Video.js ハンズオン 社員例
## 内容
ハンズオンのコードを参考にして社員が制作した例です。
動画切り替え機能を追加しています。
## ディレクトリ構成
```
.
├── README.md # このreadme
├── favicon.ico
├── index.html
├── index.js
├── style.css
├── captions        # 字幕関連
│   ├── ...
│   └── ...
├── const        # ファイル名などのデータ
│   └── videoMetaData.js
├── forthumbnail        # 動画サムネイルを作成するための資材格納ディレクトリ
│   ├── ...
│   └── ...
├── modules        # video.js の外部ライブラリ
│   ├── ...
│   └── ...
├── plugin        # 作成したプラグイン
│   └── skipTime.js
├── posters        # 再生前に表示される画像
│   ├── poster1.jpg
│   └── poster2.jpg
└── thumbnails        # 動画サムネイル
    ├── ...
    └── ...
```
